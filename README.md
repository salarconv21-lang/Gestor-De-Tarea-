# ✦ TaskFlow — Gestor de Tareas

Aplicación web de gestión de tareas construida con HTML, CSS y JavaScript puro. Sin dependencias externas.

## 🚀 Funcionalidades

- ➕ Agregar tareas con nombre y prioridad (Alta / Media / Baja)
- ✅ Marcar tareas como completadas
- 🗑️ Eliminar tareas individualmente
- 🔍 Filtrar por: Todas, Pendientes, Completadas
- 🧹 Limpiar todas las tareas completadas de un solo click
- 💾 Persistencia con `localStorage` (las tareas se guardan en el navegador)
- 📅 Fecha de creación en cada tarea

## 📁 Estructura del proyecto

```
gestor-de-tareas/
├── index.html   # Estructura de la aplicación
├── style.css    # Estilos y diseño
├── app.js       # Lógica en JavaScript
└── README.md    # Este archivo
```

## 🛠️ Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/gestor-de-tareas.git
   ```
2. Abre `index.html` en tu navegador.

¡No necesitas instalar nada!

## 💡 Tecnologías

- HTML5
- CSS3 (variables, animaciones, flexbox)
- JavaScript ES6+ (localStorage, DOM manipulation)
